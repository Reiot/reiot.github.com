Bulk Delete Entities of Lost Model in Code

Sep 27th, 2010

class EnemyUnitGroup(db.Model): pass

db.delete(EnemyUnitGroup.all(keys_only=True)[:500])

이미 클래스가 없어진 경우에는 위와 같이 임시로 선언한 다음, remote api 에서 삭제할 수 있다.

한 순간에 최대 500개의 엔티티를 바꿀 수 있다.

이런 류의 bulk delete 에는 map reduce 가 가장 적합하다.
