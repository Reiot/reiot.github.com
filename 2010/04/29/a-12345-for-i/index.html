리스트를 역순으로 순회하는 방법

a = [1,2,3,4,5]
for i in reversed(xrange(0,len(a)):
    print a[i]

list.reverse() 는 inplace 로 뒤집지만 return None 이 되므로 for in list.reverse(): 는 동작하지 않는다. range() 는 실제로 리스트 값을 리턴하지만, xrange 는 순회를 위한 이터레이터 객체를 리턴한다. reversed 역시 값 대신 이터레이터 객체를 리턴한다.
