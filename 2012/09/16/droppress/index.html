DropPress 2012-09-16

DropPress는 내가 파이썬으로 직접 구현한 간단한 정적 사이트 생성기다.

왜 이름이 DropPress이냐 하면, 드랍박스에 저장된 마크다운 포맷의 글을 ACE 에디터로 편집하고 github로 배포하는 웹서비스를 구상했기 때문이다. 드랍박스에 글을 저장해두면, 버전관리를 안해도 되고, 드랍박스와 연동되는 수많은 마크다운 에디터가 널려 있으니, 나는 변환과 배포를 구현하기만 하면 된다고 생각했다. 그런데, 비슷한 웹 에디터나 호스팅 서비스도 이미 존재했고, 또 ACE 에디터가 손대기 애매해서 포기하고, 스크립트만 만들기로 마음을 바꿨다.

기능 소개

  • 페이지 네비게이션, 아카이브, 카테고리, 태그, RSS, 최근 글목록 등 블로그의 기본 기능
  • disqus & google analytics 연동
  • 글 제목 검색
  • 쉘 명령 지원: Octopress처럼 init, generate, clean, deploy, new_post 같은 쉘 명령을 지원한다. new_post 의 경우 새 파일을 만든 후 서브라임 텍스트2에 띄워준다. 원래는 Makefile을 쓰려다가 파일 갯수를 줄인다는(?) 취지에서 그냥 파이썬으로 모두 구현해 버렸다.

구현

파이썬 스크립트 하나와 몇 개의 HTML 템플릿을 다음과 같은 라이브러리들을 짜집기해서 만들었다. 즉, 트위터 부트스트랩 덕분에 그나마 봐줄만한게 나온 셈이다.

라이브러리 적용
twitter bootstrap HTML 레이아웃 및 웹 디자인. 페이지 곳곳마다 진짜 열심히 사용했다.
bootstwatch 부트스트랩의 무료 테마. 현재 readable 테마를 사용중이다.
python-markdown 마크다운 변환. 테이블을 지원하는 플러그인도 존재한다. :)
jinja2 파이썬용 HTML 템플릿 라이브러리
yaml 설정 파일 및 글의 헤더 포맷
feedgenerator atom.xml 생성용
google-code-prettifier 자바스크립트 코드 하일라이팅

locally | 테스트용 로컬 웹서버(node.js 필요)
