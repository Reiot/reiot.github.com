페이스북 User 데이터 저장 정책 - Uid, Email, Gid, P…

Apr 7th, 2010

페이스북 user 데이터 저장 정책 - uid, email, gid, page_id, profile_update_time 무기한으로 명시된게 아니면 24시간 이상 저장할 수 없다. - 각 id 간의 관계는 저장할 수 없다. 오직 user-network 만 저장 가능. - 즉 이름, 성별, 프로필사진 URL 등 개인을 구분할 수 있는 모든 데이터는 무기한 저장 불가. - 사용자가 해지할 경우 user id 이외의 모든 데이터는 삭제해야 한다. userid 만큼은 무기한 가지고 있다가 다시 돌아올 경우 또는 ban 한 경우를 대비한다. - 인터넷이 안되거나 문제가 생긴 경우를 위해서, 나머지 데이터는 24시간 이상 유지하되 개별 앱 서버가 아니라 클라이언트에 저장할 수는 있다.
