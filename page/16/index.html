OverlayView 를 floatPane 에 두면, 이게 Overlay…

Apr 22nd, 2010

OverlayView 를 floatPane 에 두면, 이게 overlayMouseTarget 레이어보다 높기 때문에 mouseover 시 간섭이 발생한다. (일명 커서 떨림 현상) 해결책은 - overlayMouseTarget 보다 낮은 레이어에 넣으면 간섭이 발생하지 않는다. - 즉, 마커에 따라다니는 라벨 같은 오버레이들은 overlayImage 레이어에서 다 넣고, z-index 를 가지고 조절한다 - 그래도 높이 조절이 안되면, floatShadow 에 넣어버린다.

bulkloader.py 또는 appcfg.py 로 데이터를 익스포트(다…

Apr 15th, 2010

BadRequestError: app AAA cannot access app BBBs data bulkloader.py 또는 appcfg.py 로 데이터를 익스포트(다운로드) 할 때 디폴트 구현인 keyname 대신 key 자체를 저장하는 방식(-_-)을 선택한 경우, appid 가 다른 datastore 로 업로드(임포트)할 때, 기존 키값이 존재할 경우 받아들일 수 없게 된다. 특히나 sql3 로 덤프받은 데이터 역시 호환이 되지 않는다. 해결책은 - CSV 로 바꾸고, 키값은 모두 삭제해서 자동 생성 되도록 한다. 이때의 문제는 reference key 가 문제가 된다. 손쉬운 해결책은 keyname 을 쓰는 건데 이것도 못할 짓이다. - 모든 키를 돌면서 appid 부분을 교체하는 것도 하나의 해결책이다. ==> 존재하지 않는 키는 받아들이지 않는다. 오직 keyname 만 받아들임. ==> 그래서 CSV 내보내기할 때 keyname 만 내보내는 구만.
